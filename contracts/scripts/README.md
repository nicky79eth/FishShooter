An on-chain mini fish shooting game built for the Somnia ecosystem.

This project demonstrates:
- On-chain game loops
- Player resource management
- Event-driven design for analytics
- Clean and extensible Solidity architecture

## Gameplay
- Players use energy to shoot fishes
- Each fish has HP and reward points
- Killing fishes increases player score
- All state is stored on-chain

## Why this project
This repository focuses on correctness and clarity rather than UI.
It is designed to be extended with:
- Off-chain indexers
- Leaderboards
- NFT-based fish or weapons
